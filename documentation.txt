REQUISITOS PARA O BOT

1. Python 3.11 ou superior
   - Download: https://www.python.org/downloads/
   - Recomendado: Python 3.11.8 (última versão estável)

2. FFmpeg
   - Download: https://github.com/BtbN/FFmpeg-Builds/releases
   - Versão recomendada: ffmpeg-master-latest-win64-gpl-shared
   - Instruções:
     1. Baixe o arquivo .zip
     2. Extraia na pasta libraries/
     3. O caminho deve ficar: libraries/ffmpeg-master-latest-win64-gpl-shared/bin/ffmpeg.exe

3. Bibliotecas Python (instalar via pip):
   pip install -r requirements.txt

   Ou instalar individualmente:
   - discord.py[voice] 2.3.2
   - PyNaCl 1.5.0
   - yt-dlp 2024.3.10

4. Estrutura de pastas necessária:
   projeto/
   ├── main.py                # Ponto de entrada
   ├── doc.txt               # Este arquivo
   ├── src/                  # Código fonte
   │   ├── classes/         # Classes do sistema
   │   │   ├── __init__.py
   │   │   └── music_manager.py
   │   ├── config/          # Configurações
   │   │   └── config.py
   │   └── amadeus_neural_network.py
   └── libraries/           # Dependências externas
       └── ffmpeg-master-latest-win64-gpl-shared/

5. Configuração:
   - Edite o arquivo src/config/config.py
   - Adicione seu token do Discord em BOT_TOKEN

6. Configuração do YouTube (para evitar bloqueios):
   Opção 1 (Recomendada):
   - Instale a extensão "Get cookies.txt" no Chrome
   - Faça login no YouTube no seu navegador
   - Vá para youtube.com
   - Clique na extensão e exporte os cookies
   - Salve o arquivo como cookies.txt na pasta raiz do projeto

   Opção 2 (Alternativa) - Atualmente Aplicada:
   - O bot tentará usar configurações alternativas
   - Usa formato de áudio direto
   - Evita download de vídeo
   - Usa configurações otimizadas para streaming

7. Execução:
   - Abra o terminal na pasta raiz do projeto
   - Execute: python main.py

8. Comandos disponíveis:
   /music [nome/url] - Toca uma música
   /stop - Para a música e limpa a fila
   /pause - Pausa a música atual
   /resume - Retoma a música pausada
   /skip - Pula para a próxima música
   /queue - Mostra a fila de músicas

Observações:
- Certifique-se de que o FFmpeg está no caminho correto
- O bot precisa ter permissões de:
  * Enviar mensagens
  * Gerenciar mensagens
  * Conectar-se a canais de voz
  * Falar em canais de voz
- O usuário que executa os comandos precisa estar em um canal de voz
